# Recipe Shelf v1

A clean personal recipe reader for collected favorite recipes.

Recipe Shelf v1 is focused on making saved recipes easy to browse, read, and cook from without the clutter of typical recipe sites.

## Focus

- Fast recipe browsing
- Readable cooking view
- Search and category filters
- Light/dark mode
- Mobile-friendly navigation
- No clutter, ads, videos, or distractions

## Highlights

- Personal recipe library with a calm shelf-style home screen.
- Full-text search across recipe titles, tags, ingredients, and steps.
- Tag filters for quickly narrowing the collection by category.
- Clean recipe pages with subtle metadata, ingredients, steps, notes, and source links.
- Ingredient checkboxes for keeping track while preparing food.
- Cooking mode with large step-by-step instructions, previous/next controls, keyboard navigation, and progress display.
- Stay-awake control for supported browsers so the screen can remain on while cooking.
- Light and dark themes with a saved theme preference.
- Responsive layout that works on desktop and phone screens, including collapsible recipe navigation.
- Local static app design with generated JSON recipe data.

## Included

- Recipe text conversion script for normalizing local recipe notes into app data.
- Tiny local static server for desktop and phone testing on the same network.
- Web app manifest for standalone-style installation support.

## Notes

This first release is intentionally quiet and practical: it prioritizes personal use, fast access, and comfortable reading over social features, feeds, comments, tracking, or media-heavy recipe pages.
